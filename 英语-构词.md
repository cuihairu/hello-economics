## 四大构词类型

词根 - 单词的核心词义
前缀 - 改变词义
后缀 - 改变词性

|        构词                 |  
|:----------------------:|
| 前缀 + 词根             | 
| 词根 + 后缀             |
| 前缀 + 词根 + 后缀 |
| 单词 + 单词            |
